# 📊 Sales Forecasting using Machine Learning

## 🚀 Project Overview
This project focuses on predicting future sales using historical data from Walmart stores. The goal is to build a machine learning model that can forecast weekly sales and help businesses make data-driven decisions.

---

## 📁 Dataset
The dataset used in this project is **Walmart Sales Data**, which contains:

- Store number
- Date
- Montly sales
- Holiday Flag
- Temperature
- Fuel Price
- CPI (Consumer Price Index)
- Unemployment Rate

---

## ⚙️ Technologies Used

- Python 🐍
- Pandas
- NumPy
- Matplotlib / Seaborn
- Scikit-learn
- Jupyter Notebook

---

## 📌 Project Workflow

1. **Data Collection**
   - Load dataset (`Walmart.csv`)

2. **Data Preprocessing**
   - Handle missing values
   - Convert date format
   - Feature selection

3. **Exploratory Data Analysis (EDA)**
   - Visualize trends
   - Analyze seasonal patterns
   - Compare sales across stores

4. **Model Building**
   - Train machine learning models (Linear Regression / Random Forest)

5. **Model Evaluation**
   - Evaluate using metrics like:
     - Mean Absolute Error (MAE)
     - Mean Squared Error (MSE)

6. **Prediction**
   - Forecast future sales

---

## 📈 Output

- Sales prediction graphs
- Model accuracy results
- Insights into sales trends
